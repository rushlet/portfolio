Sweet Deliveries

This the website I created for a start up business selling sweet selection boxes online.
I built this website using WordPress and WooCommerce to make it easy for my client to update herself. WooCommerce also provided a lot of features to work with that we needed to run a successful ecommerce site - such as a product catalogue intergrated with shipping.
To make it more appropriate for Sharmaine's needs I added a range of plugins and custom code.
The end result was a responsive and customised website built in a week which she was extremely satisified with.
