Interactive Book Club

This is a piece of coursework I produced for my Web Technologies module.
It is a fully automated, personalised and interactive book club, built using AJAX, PHP and MySQL.
Users are able to create an account, browse through the books, read and submit reviews for books and see their account details. I created the website framework using Skeleton so it would be responsive.
Overall, I found this piece of coursework challenging yet very satisfying and recieved an A+ for it.
